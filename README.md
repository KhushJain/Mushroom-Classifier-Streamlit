# Mushroom-Classifier-Streamlit
Web app which classifies a mushroom as edible or poisonous.

### Libraries:
- Streamlit
- Scikit-learn
- Numpy
- Pandas
- Matplotlib

### To start the app:
```
streamlit run app.py
```
